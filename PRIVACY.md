# 隐私政策

## 划词翻译插件隐私政策

**最后更新日期：2026年3月20日**

### 数据收集

划词翻译插件**不收集任何用户个人数据**。

### 数据处理

1. **翻译内容**：您选中的文本会发送到 Google Translate 或 MyMemory 翻译服务进行翻译，但这些数据不会被我们存储或记录。

2. **设置信息**：您的偏好设置（如目标语言、主题等）仅存储在您的浏览器本地，不会上传到任何服务器。

3. **使用统计**：我们不收集任何使用统计信息。

### 权限说明

插件需要以下权限：

- **storage**：用于保存您的设置偏好
- **activeTab**：用于在当前页面显示翻译弹窗
- **scripting**：用于注入翻译功能到网页

### 第三方服务

本插件使用以下第三方翻译服务：

- **Google Translate API**：用于提供翻译服务
- **MyMemory API**：作为备用翻译服务

这些服务可能有各自的隐私政策，请查阅：
- [Google 隐私政策](https://policies.google.com/privacy)
- [MyMemory 服务条款](https://mymemory.translated.net/useterms.php)

### 联系我们

如有任何隐私相关问题，请通过 GitHub Issue 联系我们。
